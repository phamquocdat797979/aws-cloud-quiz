# Module 11: Automating Your Architecture Quiz - AWS

---

**Câu 1:** Which are reasons to use automation to provision resources? (Select TWO.)

A. Greater expense with manual processes

B. Automation requirement for creating some resources

C. Alignment with the reliability design principle

D. Automation requirement for high availability

E. Lack of version control with manual processes

**Đáp án: C. Alignment with the reliability design principle và E. Lack of version control with manual processes**

---

**Câu 2:** Which are benefits of using infrastructure as code (IaC) over manual processes? (Select TWO.)

A. Automate system-wide security scans.

B. Protect environments from deletion.

C. Propagate updates from a single environment to all environments.

D. Deploy environments with configuration consistency.

E. Manage all account users.

**Đáp án: C. Propagate updates from a single environment to all environments và D. Deploy environments with configuration consistency.**

---

**Câu 3:** A cloud architect wants to quickly set up a secure implementation of an Amazon FSx for Windows File Server that follows AWS best practices. Which solution should they use?

A. An AWS CloudFormation template that was downloaded from the internet

B. AWS CloudFormation Designer

C. An AWS Quick Start

D. An Amazon Machine Image (AMI) on AWS Marketplace

**Đáp án: C. An AWS Quick Start**

---

**Câu 4:** What is Amazon Q Developer?

A. A set of automated reference architectures

B. An artificial intelligence (AI)-powered coding companion

C. A template for rapid application deployment

D. An integrated development environment (IDE)

**Đáp án: B. An artificial intelligence (AI)-powered coding companion**

---

**Câu 5:** Which are reasons to use Amazon Q Developer? (Select TWO.)

A. Write compliance tests.

B. Enhance application security.

C. Automate for high availability.

D. Share open-source code.

E. Accelerate coding tasks.

**Đáp án: B. Enhance application security và E. Accelerate coding tasks.**

---

**Câu 6:** What is AWS CloudFormation?

A. A description of best practices for designing an AWS implementation

B. A template that describes your infrastructure

C. A package of all the information that is needed to launch an Amazon EC2 instance

D. An AWS service that you can use to create, model, and manage AWS resources

**Đáp án: D. An AWS service that you can use to create, model, and manage AWS resources**

---

**Câu 7:** What is AWS CloudFormation Designer?

A. A collection of reusable templates

B. A source code repository for AWS CloudFormation templates

C. A tool for automating deployments

D. A graphical design interface for creating AWS CloudFormation templates

**Đáp án: D. A graphical design interface for creating AWS CloudFormation templates**

---

**Câu 8:** Which option can be used to accomplish deployment-specific differences in an AWS CloudFormation template?

A. Use change sets.

B. Use AWS CloudFormation Designer.

C. Use conditions.

D. Use drift detection.

**Đáp án: C. Use conditions.**

---

**Câu 9:** Which option is a good way to preview changes before implementing them in AWS CloudFormation Designer?

A. Visually inspect the template.

B. Run Detect Drift.

C. Create a change set.

D. Run Update Stack.

**Đáp án: C. Create a change set.**

---

**Câu 10:** Which option is a good way to know which resources in an application environment were manually modified if the environment was created by running an AWS CloudFormation stack?

A. Run drift detection on the stack.

B. Run a change set on the stack.

C. Run a comparison in AWS CloudFormation Designer on the stack.

D. Run conditions on the stack.

**Đáp án: A. Run drift detection on the stack.**
